# Titanic Survival Analysis 🚢

## About
Exploratory Data Analysis (EDA) on the Titanic passenger dataset
to answer the question: What kind of passengers were more likely to survive?

## Key Findings
- **Gender** was the biggest factor — females had 74% survival rate vs 19% for males
- **Passenger class** was critical — 1st class had 63% survival vs 24% for 3rd class
- **Combined** — 1st class females had 97% survival, 3rd class males only 14%
- **Age** had a small effect — survivors were slightly younger on average
