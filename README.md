# Netflix Data Cleaning 🎬

## About
Data cleaning project on the Netflix Movies and TV Shows dataset
using Python and Pandas.

## What was cleaned
- Handled missing values in 6 columns
- Split duration column into duration_value and duration_unit
- Converted date_added to proper datetime format
- Exported cleaned dataset to new CSV

## Results
- Original: 8807 rows, 12 columns
- Cleaned: 8807 rows, 14 columns

- Python
